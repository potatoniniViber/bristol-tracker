# 🚽 布里斯托探员 · Bristol Stool Tracker

> 基于布里斯托大便分类法（Bristol Stool Scale）的日常肠道健康记录工具。纯前端，数据只存本地，无需注册，用完即走。

---

## 💡 灵感来源

减肥期间只关注热量摄入，直到偶然接触到 **布里斯托大便分类法**——1997年由英国布里斯托大学的 Ken Heaton 和 Stephen Lewis 医生提出，将人类粪便归纳为 7 种标准形态，至今仍是医学界评估肠道传输功能的重要参考。

看完觉得：**这么科学直观的东西，居然没有对应的日常记录工具？** 于是花了一个下午用纯前端写了一个。

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| **每日记录** | 7种形态一键点选，支持同一天多次追加，可标记“今日无排便” |
| **实时评分** | 根据当日最差类型给出健康评分和饮食建议 |
| **近7天小结** | 自动生成最近7天的健康简评 |
| **月历视图** | 哪天拉了、拉了几次、状态如何，一眼看清，支持补录 |
| **月度分析** | 从首次记录日算起，自动生成覆盖率、健康占比和建议 |
| **隐私优先** | 所有数据存储在 localStorage，无服务器、无跟踪 |

---

## 🧠 科学依据

布里斯托分类法的核心原理：**💩在肠道停留时间越长，水分被吸得越多，形态越干硬；停留越短，越稀软。**

| 类型 | 形态 | 含义 |
|:---:|------|------|
| Type 1 | 干硬小球 | 严重便秘 |
| Type 2 | 凹凸香肠 | 轻度便秘 |
| Type 3 | 香肠状有裂纹 | 较理想 |
| Type 4 | 光滑香蕉状 | **最佳** |
| Type 5 | 软块面团 | 偏软 |
| Type 6 | 糊状 | 腹泻前兆 |
| Type 7 | 水状 | 严重腹泻 |
